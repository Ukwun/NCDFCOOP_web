## ⚡ QUICK COMMANDS

### Test Locally (Development)
```bash
cd c:\development\coop_commerce_web
npm install
npm run dev
```
Then open: http://localhost:3000

### Build for Production
```bash
npm run build
npm start
```

### Deploy to Netlify
1. Push to GitHub
2. Open netlify.com
3. Click "New site from Git"
4. Select repository
5. Build command: `npm run build`
6. Publish directory: `.next`
7. Click Deploy!
